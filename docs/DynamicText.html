Class: DynamicText

DynamicText(element, play_mode, autoreset)

new DynamicText(element, play_mode, autoreset)

DynamicText es una clase que permite manipular el texto de un elemento HTML, consiste en un texto que se descompone en caracteres y se manipula para crear efectos visuales
Parameters:
Name Type Default Description
element * Elemento HTML que contiene el texto
play_mode * loop Opcional, loop o oneshot
autoreset * false Opcional, permite reiniciar automaticamente
